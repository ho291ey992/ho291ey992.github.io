Markdown 語法

By Steve 2025-01-10

基礎語法

  • 標題大小：markdown設定上分為6個大小，透過增加 # 設定不同大小標題(1h 最大、6h最小)。

    # h1 標題
## h2 標題
### h3 標題
#### h4 標題
##### h5 標題
###### h6 標題

    備註：若是想要修改標題文字大小、顏色，可以在scss檔中搜尋的是否有類似語法。

    .content {
 h1{
   font-size: 2.5rem;
   font-weight: bold;
   color: #333;
   },
 h6 
 }

  • 粗體

    **粗體**
__粗體__

  • 斜體

    *斜體*
_斜體_

  • 網底

    `網底 `

  • 刪除線

    ~~刪除線~~

  • 換行

    <br>

  • 引用

    引用內容

    > 引用內容

  • 轉義符號：*這樣不會變成粗體或斜體*

    \*這樣不會變成粗體或斜體\*

  • 註解：<!– 插入註解 –> 。

    <!-- 在這之中的內容都是不會顯示的 -->

    備註：瀏覽器中開啟 開發者模式(F12) 會被看到註解。



表格、插入空白

  • 表格

    • —：向左對齊 (預設)
    • :—：向左對齊
    • —:：向右對齊
    • :—:：置中對齊
    向左對齊 向左對齊 向右對齊 置中對齊
    … … … —
    … … … — 
    | 向左對齊| 向左對齊 | 向右對齊 | 置中對齊 |
| --- | :--- | ---: | :---: |
| ... | ... | ... | --- |
| ... | ... | ... | --- |

  • 在 Markdown 文檔中不能直接插入空白，要使用 HTML 標記插入空格：
    •   一格
    •   四格 
    &nbsp; 一格
&emsp; 四格
