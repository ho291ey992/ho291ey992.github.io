conda 環境介紹

By Steve 2025-08-19

conda

  • 檢視還進
conda env list

  • 建立環境
conda create -n <新環境名> python=<版本>
conda create -n my_env_name python=3.12

  • 啟動環境
conda activate  <環境名>
conda activate my_env_name

  • 刪除環境
conda remove -n my_env_name --all

  • 安裝套件
conda install pandas

  • 離開環境
conda deactivate

  • 複製虛擬環境
conda create -n <新環境名> --clone <舊環境名>
conda create -n new_env_name --clone old_env_name

  • 修改虛擬環境名稱
conda rename -n <舊環境名> -d <新環境名>
conda rename -n old_name -d new_name

  • 刪除虛擬環境
conda env remove -n <環境名> --all
conda env remove -n my_env --all

environment.yml

  • 輩分環境
conda env export <檔案位置>\<檔案名>
conda env export > D:\environment.yml 

conda env export -n <環境名> > <檔案位置>\<檔案名>
conda env export -n my_env_name > D:\test.yml

  • 還原環境
conda env create -n <環境名> -f <檔案位置>\<檔案名>
conda env create -n my_env_name -f D:\environment.yml
